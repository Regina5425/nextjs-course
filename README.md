### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Regina5425/nextjs-course
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the app on your localhost
   ```js
   npm run dev
   ```
